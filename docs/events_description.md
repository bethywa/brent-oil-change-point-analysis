# Event Dataset Description

This document describes the geopolitical, economic, and policy events compiled
to contextualize Brent oil price movements.

## Data Source
Events were collected from publicly available historical records, including:
- OPEC announcements
- Major geopolitical conflicts
- Global economic crises

## Columns
- date: Event start date
- event_name: Short title
- description: Detailed explanation
- region: Affected region
- impact_level: Qualitative impact estimate

## Intended Use
The event dataset is used for **contextual interpretation only** and does not
imply causal attribution.
